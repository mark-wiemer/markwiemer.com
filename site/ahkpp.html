AHK++

AHK++ (formerly AutoHotkey Plus Plus) is an AHK language support extension for AHK v1 and v2, supporting IDEs like VS Code, VSCodium, and Theia IDE. AHK++ provides syntax highlighting, code formatting, in-editor debugging, optional lint warnings, IntelliSense, and more.

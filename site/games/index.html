Sometimes I make little games :)

  • Snake (experimental!): A little toy snake game. He moves pretty fast and only stops when you press Escape to enter debug mode. Press Space to move him forward one bit. Use WASD or arrow keys to turn. No mobile support yet. Refresh to restart. No apples or points or game over, just a little snake crawling around for now :) Originally built following How to Build a Snake Game In JavaScript by Fakorede Damilola for freeCodeCamp, I'm rewriting it to support new features.

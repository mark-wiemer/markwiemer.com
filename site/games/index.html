Games

Sometimes I make little games :) For Racetrack Tycoon and Snake, press Escape to pause, R to restart, and Space to enter debug mode. While in debug mode, press Space to tick the game forward. Press Escape to exit debug mode.

Games only work with a keyboard right now, mobile support coming... eventually!

  • Dodge the Creeps: A simple game made using Godot, not original to this site by any means! Exported from Hello Godot, source code coming to this repo soon.
  • Racetrack Tycoon: Customize your racetrack, hire a fleet of racers, and entertain massive crowds! Original concept, extremely early in-develop­ment. Use WASD or arrow keys to accelerate your car. No friction yet!
  • Snake: Eat apples to grow, how big can you get? Use WASD or arrow keys to turn.
